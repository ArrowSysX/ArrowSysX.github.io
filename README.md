# ArrowSysX.github.io

## how to compile docs:

### windows:
npm install -g honkit </br>
md {compile dir name} </br>
git clone https://github.com/ArrowSysX/ArrowSysX.github.io </br>
cd docs/ </br>
honkit build </br>

### linux/macos:
npm install -g honkit </br>
mkdir {compile dir name} </br>
git clone https://github.com/ArrowSysX/ArrowSysX.github.io </br>
cd docs/ </br>
npx honkit build </br>
